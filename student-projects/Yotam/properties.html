Demonstrating CSS Properties

.color

Sets the text color to red.

.font-size

Sets the font size to 16 pixels.

.margin

Sets the margin to 10 pixels.

.padding

Sets the padding to 20 pixels.

.width

Sets the width to 200 pixels.

.height

Sets the height to 300 pixels.

.display

Determines how an element is displayed.

.position

Sets the positioning method for an element.

.float

Specifies how an element should float.

.border

Sets the border properties.

.text-align

Sets the horizontal alignment of text.

.text-decoration

Adds decorations to text (e.g., underline, strikethrough).

.transition

Adds transition effects to elements.

.box-shadow

Adds a shadow effect to elements.

.opacity

Sets the opacity of an element.

.font-family

Specifies the font family for text.

.text-transform

Controls the capitalization of text.

.border-radius

Adds rounded corners to elements.

.overflow

Controls how content overflows an element's box.

.background-image

Sets the background image of an element.

.line-height

Sets the height of a line of text.

.box-sizing

Determines how the width and height of an element are calculated.

.z-index

Specifies the stacking order of positioned elements.
